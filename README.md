# OpsLog - Modern IT Operations Logbook SaaS

OpsLog adalah platform manajemen logbook operasional khusus untuk profesional IT yang ingin mendokumentasikan aktivitas harian secara terstruktur, efisien, dan profesional.

## ğŸš€ Fitur Utama

* **Modern Sidebar Layout**: Navigasi intuitif untuk akses cepat ke Dashboard, Input Log, dan Laporan.
* **Google Calendar Style**: Tampilan kalender bulan penuh yang luas untuk visualisasi aktivitas harian yang lebih jelas.
* **Gamified Productivity**: Fitur "Streak Saat Ini" untuk memantau konsistensi pengisian logbook harian.
* **Smart Log Entry**: Form pengisian aktivitas yang mendukung format Markdown dan kategori dinamis.
* **Responsive & Dark Mode Support**: Tampilan profesional yang mendukung mode gelap untuk kenyamanan mata saat bekerja.
* **Profile-Based Activity**: Setiap log dicatat sesuai dengan akun yang sedang masuk (Login Dummy System).
